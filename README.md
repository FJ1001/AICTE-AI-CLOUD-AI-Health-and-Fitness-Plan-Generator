# AI Health and Fitness Plan Generator

## Overview
AI-powered app that generates personalized diet and workout plans, combining them into a holistic health strategy based on user data.

## Key Features
- **Personalized Meal Plans:** Keto, Vegetarian, Low Carb, Balanced  
- **Custom Workout Routines:** Weight Loss, Muscle Gain, Endurance, Flexibility  
- **Holistic Health Plans:** Combines diet and fitness into one strategy  
- **Intuitive Streamlit UI:** Sidebar inputs and visual cards for easy interaction  
- **Motivational Messages & Progress Tips**  
- **Handles API Rate Limits Gracefully**  

## System Requirements
- Python 3.8+  
- Libraries: Streamlit, Agno  
- Gemini API key (Google)  
- Internet connection for AI inference  

## How to Run
1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2. Run the Streamlit app:
    ```bash
    streamlit run AI_Health_and_fitness.py
    ```
3. Input user details: name, age, weight, height, activity level, dietary preference, fitness goal  
4. Click **Generate Health Plan**  
5. View your personalized diet, workout, and holistic plan
